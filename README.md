# 💅 Egly Vitiniski | Premium Nail Studio Website

A **100% Mobile-First** landing page designed for a high-end Nail Designer professional based in Portugal. This project focuses on luxury aesthetics, smooth user experience (UX), and a direct conversion funnel via WhatsApp integration.

---
## 🌐 View project:

🔗 **[Ver Projeto Online](https://mrvitiniski.github.io/EglyVitiniski/)**

## ✨ Features

- 📱 **Mobile-First Design**: Engineered for the modern user. Optimized for speed and responsiveness on all mobile devices.
- 📅 **Custom Booking Modal**: A sophisticated modal window with a native `datetime-local` picker, replacing cluttered floating buttons for a cleaner UI.
- 📍 **One-Click Navigation**: Address linked directly to Google Maps API for instant GPS routing to the studio in Corroios.
- 🎨 **Luxury Typography**: Featuring the *Hurricane* signature font for a personalized brand feel and *Montserrat* for professional readability.
- 💬 **WhatsApp Automation**: Seamlessly pre-fills booking details (Date/Time) into the professional's WhatsApp chat.
- ⚡ **Performance**: Built with Vanilla JS and CSS3 variables—zero heavy libraries, ensuring instant loading.

## 🛠️ Tech Stack

- **HTML5**: Semantic and SEO-ready structure.
- **CSS3**: Custom properties, Flexbox/Grid layouts, and glassmorphism effects.
- **JavaScript**: Custom DOM manipulation for modal management and message formatting.
- **Google Fonts**: Hurricane & Montserrat.
- **Font Awesome**: Professional vector icons.

---

## 🚀 How it Works

1. **The Hook**: Users land on a focused "Hero" section with a high-contrast call to action.
2. **The Decision**: The service menu provides clear pricing and expandable details to keep the interface clean.
3. **The Conversion**: The "Agendar Agora" (Book Now) button triggers a modal where the user selects their preferred slot.
4. **The Connection**: Upon confirmation, the system formats the booking request and opens WhatsApp with the message ready to send.

   ```bash

   ```
